animated-anchros
================

jQuery plugin for extending a browsers anchor technic, like offsets, animation and right positioned hash-tags


/* =============================================================

 *
 *      // HOW TO USE
 *
 * Example :
 *      $('.anchor').animatedanchor(); 
 *
 * Options:
 *     
 *      Available options and their defaults:
 *
 *      offset:114,
 *      delay:1000,
 *      easing:"linear",
 * 
 *      $('.anchor').animatedanchor({option1, 'value'});
 *
 *
 * Methods:
 *     scrollTo(element)
 * ============================================================== */
